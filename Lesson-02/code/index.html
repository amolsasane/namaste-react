#Parcel is the Beast Features

- To make our app production ready we have to minify, optimize, clean consoles, remove unused code etc..

- To achieve the same we use BUNDLERS: Bundler is a package/dependency which we use to optimize our code there generally 3 types of bundlers

- Ex: Parcel, Vite, WebPack

- Mostrly all bundlers do the same job.

- If we want a packages in our app we have to use a package manager called 'npm' or 'yarn'
